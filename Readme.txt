para la ejecucion del programa se debe proceder de la siguiente manera:
1. Se crea un directorio donde vamos a colocar los siguientes archivos:
- Bases nominales con el nombre: NEFRO_202409.xlsx 
- Bases de medicamentos con el nombre: MEDS_202410.xlsx
- Archivo con los codigos de las EPS : 0_EPS.txt (este archivo es el mismo a menos que se relize alguna actualizacion de EPS)
- Archivo con el codigo de python: CODE_01.py
2. Debemos ubicarnos en el directorio y se abrimos una terminal en en la que vamos a ejecutar el codigo de python: python CODE_01.py
3. Posteriormente a la ejecucion del codigo se van a generar dos archivos .txt los cuales ya tienen la informacion mas actualizada de las bases nominales 
4. Los archivos generados son los insumos que se van a utilizar para el tablero de indicadores
